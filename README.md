# Kenkaru
健康を手軽に気軽に『KENKARUケンカル』

## How to build
### Flutter
- Install Flutter SDK https://docs.flutter.dev/get-started/install
- change directory flutter/ and build it

### PocketBase
- Open DevContainer
- `go run . serve` or `go build'` and `./myapp serve`
- https://pocketbase.io/docs/go-overview/
